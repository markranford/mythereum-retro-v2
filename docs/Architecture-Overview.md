# Architecture Overview: Mythereum Retro V2

## System Architecture

Mythereum Retro V2 is built on a modern, scalable architecture leveraging the Internet Computer's unique capabilities for decentralized gaming.

## Frontend Architecture

### Technology Stack
- **Framework**: React 19 with TypeScript
- **Styling**: TailwindCSS with custom OKLCH color system
- **Routing**: TanStack Router for type-safe navigation
- **State Management**: React Query for server state, React Context for UI state
- **Authentication**: Internet Identity integration
- **UI Components**: Shadcn/ui component library
